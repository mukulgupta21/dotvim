# How to use?

## Dependency
One of the plugins uses `ack` which may not be installed by default.
Install it using:
```
sudo apt-get install ack
```
## Installation
```
cd ~
git clone git@github.com:mukulgupta21/dotvim.git .vim
ln -s ~/.vim/vimrc ~/.vimrc
cd ~/.vim
git submodule init
git submodule update
```
